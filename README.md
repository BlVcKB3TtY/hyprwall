## üé® Using hyprwall

After installing hyprwall, you can easily set your wallpapers. Here‚Äôs how:

1. **Launching hyprwall:**
   - Open your terminal.
   - Type `hyprwall` and press Enter.

2. **Selecting Wallpapers:**
   - Use the command `hyprwall set <path-to-image-or-video>` to set a wallpaper.
   - You can set both image and video formats.

3. **Auto Modes:**
   - hyprwall supports auto modes, allowing you to change wallpapers based on your preference. Use the command:
   ```
   hyprwall auto <mode>
   ```
   Replace `<mode>` with your desired setting (like time-based or random).

4. **Checking Status:**
   - To see the current wallpaper, just type:
   ```
   hyprwall status
   ```

5. **Exiting:**
   - To exit hyprwall, simply type `exit` or close the terminal window.
